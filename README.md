Getting Started

1. Navigate to:
   dist / Pixelith / run.exe

2. Double-click run.exe to launch Pixelith.

Note:
Some antivirus software may display a warning due to the application being
bundled. This is a known false positive. You may safely choose Ignore or Allow.

--------------------------------------------------

How to Use

- Click "+ New Project" to create a new project
- Open a project from the Projects screen
- Draw using the Pen Tool
- Change colors using the Color Picker

Your projects are automatically saved.
